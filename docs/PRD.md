# ברידג' גליל — פלטפורמת הלימוד של נפתלי קריספיס
## PRD + אסטרטגיה + ארכיטקטורה (מסמך מאוחד)

### 1. חזון
פלטפורמת לימוד ברידג' בעברית, פרימיום, לקהל 55+ בגליל המערבי ובכל הארץ:
אתר תדמית, תוכן חינמי איכותי, קורסים ברורים, תרגול אינטראקטיבי, ובסיס למנועי ניתוח.

### 2. קהל יעד
- ראשי: גמלאים ופורשים 55-80, עברית שפת אם, רמת דיגיטל בינונית. דורש: פונט גדול, ניגודיות, ניווט פשוט.
- משני: שחקני מועדון שרוצים לשפר הכרזות/משחק יד.
- שלישוני: מתחילים צעירים שמחפשים תוכן עברי ברשת (SEO).

### 3. הצעת ערך (מול מתחרים)
- gobridge: פורטל עמוס, לא מותאם מובייל היטב, אמון נמוך בעיצוב. אנחנו: נקי ופרימיום.
- bridgeonlineacademy: קורסים בתשלום, מעט תוכן חינמי. אנחנו: משפך תוכן חינמי -> קורס.
- bridgesalon: בלוג אישי, ללא תרגול. אנחנו: תרגול אינטראקטיבי בדפדפן.
פער השוק: אף מתחרה לא משלב תוכן עברי איכותי + תרגול אינטראקטיבי + מורה אזורי אמיתי עם שיעורים פרונטליים.

### 4. עמודי המוצר (Sitemap)
```
/                     דף בית (hero, הצעת ערך, מסלולים, המלצות, CTA)
/about.html           אודות נפתלי (אמון, ניסיון, גליל מערבי)
/learn/               מרכז ידע חינמי (מאמרים ושיעורים)
  lessons/basics-*    יסודות: חוקים, מכרז, משחק יד, הגנה
  articles/*          מאמרי העמקה (מוסכמות, ספירת נקודות, סיגנלים)
/courses.html         קורסים: מתחילים / ממשיכים / מתקדמים + מחירים ורישום
/practice/            תרגול חכם: חידוני הכרזות, זיהוי יד, ספירת נקודות
/contact.html         יצירת קשר + שיעורים בגליל המערבי
```

### 5. מערכת תרגול (Phase 1 — client-side בלבד)
- מנוע חידונים JS: שאלות הכרזה (מציג יד, בוחרים הכרזה, הסבר מיידי).
- מחשבון נקודות יד אינטראקטיבי (HCP + חלוקה).
- תרגול זיהוי: "מה ההכרזה הנכונה?" עם ידיים אקראיות מחוללות בצד לקוח.
- Phase 2: double-dummy solver בדפדפן (bridge-dds-js / WASM) לניתוח ידיים.
- Phase 3: שרת ניתוח (endplay/dds) + AI הכרזות (ben) — מתועד, לא נבנה עכשיו.

### 6. ארכיטקטורה
- אתר סטטי טהור: HTML5 + CSS + Vanilla JS. אפס backend, אפס build step.
- RTL מלא, lang=he, פונטים: Heebo/Assistant (Google Fonts).
- תרגול = מודולי JS עצמאיים, ידיים מיוצגות כמחרוזות PBN-like.
- פריסה: כל static host (Netlify/GitHub Pages). טפסים: mailto/WhatsApp deep-link.
- SEO: meta מלא, OG, JSON-LD (LocalBusiness + Course), sitemap.xml, עברית סמנטית.

### 7. עיצוב
- פלטה: ירוק-שולחן עמוק (#1B4D3E) + קרם (#F7F3EA) + זהב (#C9A227). לא קזינו, כן מועדון יוקרתי.
- טיפוגרפיה גדולה (base 18px+), ניגודיות AA+, כפתורים גדולים (קהל מבוגר).
- קלפים: רנדור CSS/Unicode suits (♠♥♦♣), אדום/שחור מסורתי.

### 8. הנחות (assumptions)
- מחירים: קורס מתחילים 480 ש"ח (8 מפגשים) — placeholder לעריכת נפתלי.
- טלפון/מייל: placeholders מסומנים TODO.
- אין תשלום אונליין בשלב 1; רישום דרך טלפון/וואטסאפ (מתאים לקהל).

### 9. KPI / הצלחה
- Lighthouse 90+ נגישות ו-SEO, טעינה < 1s, עובד מלא במובייל RTL.
- משפך: כתבה חינמית -> CTA קורס -> יצירת קשר.
